# SmallApp-For-QQmusic
微信小程序Demo，仿QQ音乐

**实现功能**
- 热门歌单
- TOP
- 搜索
- 播放、暂停、歌词滚动、播放进度

**接下要实现的功能**

- 单曲循环、列表循环播放
- 上、下首播放
- 音量控制
